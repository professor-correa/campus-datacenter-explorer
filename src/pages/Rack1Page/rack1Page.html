ARM, Apache e FTP

O que esse servidor faz?

Diferente dos demais Racks presentes no nosso datacenter, o Rack 1 possui uma particularidade, ele é aberto, com isso é possível ver toda a sua estrutura, além de trazer um aspecto visual diferente para o datacenter. Ele é responsável pelos serviços de armazenamento e hospedagem de dados, este rack reúne e organiza os equipamentos essenciais para que a rede funcione de forma estruturada. Dentro dele estão concentrados o servidor Apache, responsável por disponibilizar páginas webs para os usuários, e o servidor FTP, que permite a transferência e o gerenciamento de arquivos dentro da própria rede. Dessa forma, ele atua como um ponto central onde os dados são guardados, acessados e compartilhados de maneira segura e eficiente. Com esse rack podemos ver que existem diversos tipos e formatos de racks diferentes em nossa escola.

Switch

Ele é um equipamento de hardware que conecta vários dispositivos (computadores, impressoras, servidores, câmeras IP) numa mesma rede (LAN), permitindo que eles conversem entre si de forma eficiente.

Monitor

O Monitor KVM é um dispositivo que integra teclado, vídeo e mouse em uma única estrutura. Ele permite controlar vários servidores a partir de um único console, economizando espaço e facilitando a gestão.

Servidor

Um servidor é um computador, mas diferente do seu computador pessoal (onde você joga, digita textos ou navega na internet), o trabalho do servidor não é atender a uma pessoa sentada na frente dele. O trabalho dele é atender (servir) outros computadores.

Patch Panel

O Patch Panel é um painel passivo (não usa energia) cheio de portas de rede (RJ45), que serve como um ponto intermediário entre os cabos que vêm das paredes e o Switch.

Falando mais sobre armazenamento

O dispositivo de armazenamento desse rack é um equipamento usado para guardar grande quantidade de dados em um só lugar. Ele utiliza vários discos juntos para aumentar a velocidade e manter as informações seguras, mesmo se algum deles falhar. Esse tipo de sistema é muito usado em empresas e centros de tecnologia para armazenar arquivos importantes, fazer backups e manter dados acessíveis de forma rápida e confiável para todos que precisam utilizá-los na rede.

O que é SSH

Pense nele como uma porta secreta e segura para entrar em um servidor. Com o SSH, você consegue controlar o servidor à distância, digitando comandos. É como se fosse um controle remoto para computadores, mas protegido por senha e criptografia.

O que é Apache?

O Apache é um programa usado para hospedar sites. Ele recebe os pedidos que fazemos pelo navegador e envia de volta as páginas que queremos ver. Muito flexível, ele permite adicionar recursos de segurança, login, compressão e suporte a outras linguagens. Por ser estável e funcionar em vários sistemas, tornou-se um dos servidores mais populares do mundo para colocar sites no ar.

Veja a documentação técnica

FTP

O FTP é um método usado para enviar e receber arquivos entre um servidor e um computador. Ele é fácil de usar e funciona bem, mas não possui criptografia própria, o que significa que os dados podem ficar expostos se não houver outras opções de segurança junto com ele.

Documentação Técnica

Armazenamento

O dispositivo de armazenamento em um rack de redes, também conhecido como Storage ou Servidor de Armazenamento, é um equipamento dedicado a concentrar e gerenciar grandes volumes de dados de forma centralizada. Ele é composto por vários discos rígidos (HDs ou SSDs) organizados em arranjos RAID, garantindo desempenho, redundância e segurança das informações. Esse tipo de equipamento é amplamente utilizado em ambientes corporativos e data centers para armazenar arquivos, backups e bancos de dados, permitindo acesso rápido e confiável pelos servidores e usuários da rede. SSH

Definição: Protocolo de comunicação segura para acesso remoto a servidores.

Arquitetura: Software usado para conectar (ex.: OpenSSH). Servidor SSH: serviço rodando no servidor. Criptografia: RSA, AES, ou chaves públicas/privadas.

Funcionalidades: Login remoto seguro. Execução de comandos em linha de comando. Tunelamento de conexões (port forwarding). Transferência de arquivos via SCP ou SFTP.

Casos de uso: Administração de servidores Linux/Unix. Automação de tarefas remotas. Conexões seguras em redes públicas.

Apache

O Apache é um servidor web de código aberto amplamente utilizado para hospedar sites e aplicações na internet. Ele funciona como intermediário entre o navegador do usuário e o servidor que contém os arquivos do site, processando requisições HTTP e entregando páginas webs ao cliente. Altamente configurável, o Apache suporta módulos que adicionam recursos como autenticação, segurança, compressão e suporte a linguagens como PHP.
