# pages

Aqui vai ficar as pastas de cada pagina

```sh
pages/
Rack1Page/
Rack2Page
...
LandingPage/
```

Dentro de cada pasta terá os arquivos `html`, e uma pasta `styles/` 
```sh
Rack1Page/
styles/
header.html
main.html
footer.html
```

Dentro de `styles/` vai ter o arquivo css de cada sessão responsável por cada pessoa
